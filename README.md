# opencv_helloworld

Simplest OpenCV project using CMake.  
You can start OpenCV easily with this repository.  
Please edit it as you want.  

## how to use

```
git clone git@github.com:RyoheiKomiyama/opencv_helloworld.git
cd opencv_helloworld
mkdir build
cd build
cmake ..
make
./opencv_helloworld
```
